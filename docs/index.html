❓ Windows 提示"已保护你的电脑"？

这是 SmartScreen 对未签名应用的常规提示。点击「更多信息」→「仍要运行」即可。FocusForest 完全开源，代码可审查。

❓ 杀毒软件报毒？

PyInstaller 打包的程序可能被误报。FocusForest 是开源项目，源代码完全公开可审查，不包含任何恶意代码。可添加信任或从源码自行打包。

❓ 需要安装 Python 吗？

不需要。下载解压后直接运行 FocusForest.exe 即可，无需安装任何依赖。
