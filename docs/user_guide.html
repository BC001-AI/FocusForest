📥 安装与启动

下载

1打开 GitHub，搜索 FocusForest

2进入仓库页面，点击右侧 Releases

3下载最新版本的 FocusForest.zip

4解压到任意目录

5双击 FocusForest.exe 即可使用，无需安装

💡 提示：FocusForest 是绿色软件，解压即用，不写注册表，不联网。如需卸载，直接删除文件夹即可。

🚀 快速上手（3步开始专注）

1输入任务名：在专注页面顶部的输入框中，输入你接下来要专注的任务（如"写报告"、"背单词"）

2点击开始：点击「开始专注」按钮，进入倒计时

3保持专注：倒计时结束前不要操作手机或切换应用，专注完成后你将获得一棵树！

💡 默认设置：专注时长 25 分钟，短休息 5 分钟，长休息 15 分钟。可在设置中自定义。

专注计时器

专注页面的核心功能，支持以下操作：

  • 开始专注：点击按钮或按快捷键开始倒计时
  • 暂停/继续：临时暂停，继续后倒计时恢复
  • 停止：提前结束本次专注（不会种树）
  • 跳过休息：休息期间可跳过，直接进入下一轮专注
  • 任务名显示：专注期间顶部显示当前任务名，提醒自己保持专注

悬浮窗模式

专注期间点击「最小化」按钮，窗口缩小为悬浮小窗：

  • 显示剩余时间和任务名
  • 可拖动到屏幕任意位置
  • 点击可还原完整窗口
  • 专注结束自动提醒

消息中心

接收工具的通知和更新消息：

  • 全部：查看所有消息
  • 通知：系统通知类消息
  • 更新：版本更新消息
  • 点击「👁 已读」切换显示/隐藏已读消息
  • 点击「✓ 全部已读」一键标记所有消息为已读

❓ 常见问题

Q：Windows 提示"Windows 已保护你的电脑"，无法运行？
这是 Windows SmartScreen 的安全提示，因为 FocusForest 没有购买微软数字签名证书（证书费用高昂）。点击「更多信息」→「仍要运行」即可。FocusForest 完全开源安全，请放心使用。
Q：杀毒软件报毒怎么办？
这是误报。FocusForest 使用 PyInstaller 打包，部分杀毒软件会误判。解决方法：将 FocusForest.exe 添加到杀毒软件的信任列表（白名单）即可。
Q：专注被打断怎么办？
如果点击「停止」提前结束专注，本次不会种树。这是设计如此——鼓励你完整地完成每一次专注。如果只是临时需要查看信息，可以使用「暂停」功能，暂停后继续仍可种树。
Q：数据会丢失吗？
专注数据保存在本地数据库中，不会因关闭程序而丢失。但请注意不要删除程序目录下的 data 文件夹。
Q：专注时长可以自定义吗？
可以。在设置中可以自定义专注时长、短休息时长和长休息时长。
Q：FocusForest 会联网吗？
不会。FocusForest 是纯本地应用，不联网、不上传数据、不收集任何个人信息。
Q：如何更新到新版本？
关注 GitHub 仓库的 Releases 页面，下载最新版本替换旧文件即可。数据文件会自动兼容。
⚠️ 重要：FocusForest 完全免费开源，仅通过 GitHub 发布。任何其他渠道的下载均非官方版本，请注意甄别。
